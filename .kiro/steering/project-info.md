# Meeting Time Finder - Project Information

## Purpose
A single-purpose web application that helps users find the perfect meeting time across multiple time zones.

## Tech Stack
- Pure HTML/CSS/JavaScript (no frameworks)
- Responsive design
- Client-side only (no backend required)

## Key Features
- Add multiple time zones
- Visual time comparison
- Highlight business hours
- Copy meeting time to clipboard

## Development Standards
- Clean, readable code
- Mobile-first responsive design
- Accessibility compliant
- No external dependencies
